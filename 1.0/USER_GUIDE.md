# SundayOS iPhone 快捷指令配置指南

## 📱 快捷指令 #1：SundayOS 主入口

这是你每天最常用的快捷指令。通过语音或文字与 Sunday 对话。

### 配置步骤

1. 打开 iPhone「快捷指令」App
2. 点击右上角「+」创建新快捷指令
3. 按顺序添加以下操作：

| 步骤 | 操作 | 参数 |
|------|------|------|
| 1 | **听写文本** | 语言：中文（普通话），停止聆听：暂停后 |
| 2 | **设定变量** | 变量名：`用户输入`，值：上一步的听写文本 |
| 3 | **文本** | 输入：`https://你的API地址/api/chat` |
| 4 | **设定变量** | 变量名：`API地址` |
| 5 | **词典** | 点击添加：<br>• 键：`message`，值：选择变量「用户输入」<br>• 键：`user_id`，值：你的用户ID<br>• 键：`voice_input`，值：是 |
| 6 | **文本** | 输入：你的 API Key（在 .env 中的 SUNDAY_API_KEY） |
| 7 | **设定变量** | 变量名：`API_KEY` |
| 8 | **获取URL内容** | URL：选择变量「API地址」<br>方法：POST<br>请求体：JSON<br>添加新头部：<br>• `Content-Type`：`application/json`<br>• `X-API-Key`：选择变量「API_KEY」 |
| 9 | **从词典中获取值** | 词典：上一步结果<br>键：`reply` |
| 10 | **朗读文本** | 文本：上一步的 reply<br>语言：中文（普通话 - 中国大陆）<br>声音：婷婷（增强版）<br>语速：0.5 |
| 11 | **显示结果** | 文本：选择变量「reply」 |

### 添加快捷方式

- **添加到主屏幕**：快捷指令详情 → 添加到主屏幕 → 命名为 "Sunday"
- **Siri 短语**：快捷指令详情 → 添加到 Siri → 录制 "Hey Siri, Sunday" 或 "Hey Siri, 打开 Sunday"
- **Action Button**（iPhone 15 Pro）：设置 → 操作按钮 → 快捷指令 → 选择 SundayOS

## 📝 快捷指令 #2：SundayOS 记忆胶囊

快速记录想法、灵感、待办事项，Sunday 会自动帮你整理和关联。

| 步骤 | 操作 | 参数 |
|------|------|------|
| 1 | **听写文本** | 语言：中文，停止：暂停后 |
| 2 | **设定变量** | 变量名：`想法` |
| 3 | **文本** | `https://你的API地址/api/memory/store` |
| 4 | **词典** | • `user_id`：你的ID<br>• `content`：变量「想法」<br>• `memory_type`：`episodic`<br>• `importance`：`medium`<br>• `source`：`voice_capsule` |
| 5 | **获取URL内容** | POST，JSON，Header：X-API-Key |
| 6 | **朗读文本** | "已记录" |

## 🌅 快捷指令 #3：SundayOS 每日简报

每天早上自动播报天气、日程和个性化问候。可配合「自动化」在固定时间触发。

| 步骤 | 操作 | 参数 |
|------|------|------|
| 1 | **文本** | `https://你的API地址/daily-brief/你的用户ID` |
| 2 | **获取URL内容** | GET，Header：X-API-Key |
| 3 | **从词典获取值** | 键：`greeting` |
| 4 | **朗读文本** | 问候语 + 简报内容 |
| 5 | **显示结果** | 完整简报 |

### 自动化设置

快捷指令 → 自动化 → 创建个人自动化 → 特定时间 → 每天早上 7:00 → 执行「SundayOS 每日简报」

## 🎯 高级技巧

### 配合 NFC 标签
- 在床头贴 NFC 标签 → 触碰触发「每日简报」
- 在车钥匙上贴 NFC 标签 → 触碰触发「今天的路况」

### 配合专注模式
- 「工作模式」自动切换 → Sunday 知道你在工作，回复更简洁
- 「睡眠模式」→ Sunday 暂停主动提醒

### 配合日历
- 快捷指令中可读取日历事件，传给 Sunday
- 例：检测到会议 → Sunday 自动准备会议相关内容
