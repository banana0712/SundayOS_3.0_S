# 液态玻璃设计研究与规范

## 真正的Glassmorphism设计原则

### 核心特征
1. **层次透明度**：背景模糊 + 半透明 + 多层叠加
2. **柔和边界**：细腻的边框（1px，rgba(255,255,255,0.18)）
3. **微妙阴影**：轻柔的投影，避免过重
4. **流动感**：渐变、动画要自然流畅
5. **对比度控制**：文字必须清晰可读

### 典型参数范围
- **背景模糊**: 40-80px（不是越大越好）
- **透明度**: 0.05-0.15（过高会失去玻璃感）
- **边框**: 1px solid rgba(255,255,255,0.12-0.2)
- **内阴影**: inset 0 1px 0 rgba(255,255,255,0.1)
- **外阴影**: 0 8px 32px rgba(0,0,0,0.12)

### Apple设计语言要点
1. **对齐**: 所有元素垂直居中对齐（items-center）
2. **间距**: 使用4px的倍数（8, 12, 16, 20, 24）
3. **圆角**: 常用12px、16px、20px
4. **交互反馈**: scale(0.98)而不是过度动画
5. **文字**: 行高1.5，最小字号14px

## 当前问题分析

### 1. 输入框对齐问题
- **问题**: `items-end` 导致文本和按钮底部对齐
- **修复**: 改为 `items-center` 垂直居中对齐
- **问题**: padding不一致
- **修复**: 统一使用16px内边距

### 2. 液态玻璃效果不足
- **问题**: 只有静态模糊，缺少流动感
- **改进**: 添加微妙的流体动画、光泽扫过效果

### 3. 层次感不够
- **问题**: 所有元素同一层级
- **改进**: 卡片悬浮感、深度阴影、多层叠加

### 4. 颜色对比不够
- **问题**: 背景和前景对比度低
- **改进**: 增加背景饱和度或降低前景透明度

## 设计方案

### 方案A：Apple风格（推荐）
- 简洁、克制的玻璃效果
- 清晰的层次和间距
- 流畅的微动画
- 高可读性

### 方案B：流体风格
- 动态渐变背景
- 流动的光泽效果
- 类液体的形变动画
- 更强的视觉冲击
