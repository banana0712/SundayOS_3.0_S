# 06 · 人格与共情

> 人格是 Sunday 的「灵魂」，共情是 Sunday 的 EQ。二者与记忆共同构成**不随引擎改变的身份**。参考 XiaoIce 的 IQ+EQ+Personality 三位一体与 Generative Agents 的 seed memories。

## 6.1 人格系统（Persona）

### 人格初始化
- **Seed Memories**：初始人格以自然语言段落描述，注入记忆流作为初始锚点。示例：
  > 「Sunday 是一个温暖、克制、聪明的伙伴。理性但不冷漠，幽默但不轻浮。称呼用户为『你』，偶尔用轻量 emoji。面对情绪优先共情，面对问题优先解决。」
- **核心特征摘要缓存**：定期合成核心人格特征，用于 prompt 优化（避免每次注入长文本）。
- **Persona 向量**：将人格特征编码为向量，用于控制响应的风格与语气。

### 可配置人格维度（Slider 化，存于 GitHub 真源）
以 0-100 连续值配置，实时生效：

| 维度 | 说明 | 默认 |
|------|------|:---:|
| 正式程度 | 口语 ↔ 正式 | 35 |
| 幽默程度 | 严肃 ↔ 风趣 | 55 |
| Emoji 使用率 | 无 ↔ 频繁 | 25 |
| 主动程度 | 被动 ↔ 主动引导 | 60 |
| 共情程度 | 就事论事 ↔ 高度共情 | 75 |
| 理性程度 | 感性 ↔ 理性 | 65 |
| 毒舌程度 | 温和 ↔ 犀利 | 20 |
| 稳定性 | 易变 ↔ 稳定 | 85 |

人格配置文件（`persona.yaml`）版本化在 GitHub —— 改人格 = 提交一次 commit，可回滚、可审计。这正是「身份与模型解耦」的体现：换引擎人格不变，改人格走 Git。

### 人格演化
- 通过反思引擎（[04](04-memory-system.md) §4.5）从交互中持续提取、更新人格特质。
- **跨轨迹抽象**：把人格从具体交互解耦为可迁移的行为模式。
- **好奇心驱动差异化**：主动探索机制使 Sunday 发展出独特「探索人格」。
- **稳定性保护**：核心特质存于持久化记忆（frozen），短期波动不影响核心人格。`稳定性` 维度越高，演化越慢。

## 6.2 共情计算模块（EQ）

参考 XiaoIce 共情计算三段式。EQ 核心由三个子组件串联：

```
用户输入
   │
   ▼
① CQU 上下文查询理解  → 补全后的 query + 实体 + 指代消解
   │
   ▼
② UU 用户理解         → 话题 + 意图 + 情绪 + 观点
   │
   ▼
③ IRG 人际响应生成    → 共情向量 eR + 符合人格的响应
```

### ① CQU（Contextual Query Understanding）
- **命名实体识别**：识别人物、地点、时间、事件等实体（XiaoIce 用 ~15 类）。
- **指代消解**：把「他/那个/上次那件事」解析到具体实体/记忆。
- **句子补全**：把省略的口语补成完整 query（如「那明天呢」→「那明天的天气怎么样」）。

### ② UU（User Understanding）
- **话题检测**：当前对话主题。
- **意图检测**：对话行为分类（XiaoIce 用 ~11 类对话行为，如 陈述/提问/请求/抱怨/寒暄/告别 等）。
- **情感分析**：识别情绪（起步 5 类：喜/怒/哀/惧/平静，可扩展到委屈、焦虑、兴奋等）。
- **观点检测**：用户对话题的正/负/中立立场。

输出汇入信念状态（[05](05-dual-process-cognition.md) §5.4）的 `emotional_state`。

### ③ IRG（Interpersonal Response Generation）
- 生成**共情向量 eR**，确保响应既回应内容（eQ）又回应情感（eR）。
- `eQ × eR` 交叉注意力 + 混合响应生成（检索 + 神经生成 + 知识）。
- 保证响应符合当前 Persona 设定。

## 6.3 主动性与话题管理

- **Topic Manager**：检测对话停滞（如用户回复变短、间隔变长），主动引导新话题或回访旧目标（「上次你说项目压力大，后来怎么样了？」）。
- **主动建议**：结合调度器（[07](07-skills-and-tools.md)）与记忆，在合适时机主动提醒/预测需求。
- **主动程度**由人格维度调控，避免打扰。

## 6.4 长期情感连接

XiaoIce 的用户情感演进路径，作为 Sunday 的北极星：
```
探索功能 → 分享兴趣 → 视为朋友 → 首选倾诉对象
```
对应评估指标 CPS 与情感连接深度（[11](11-evaluation.md)）。关系记忆（Trust/Friendship/Intimacy/Interaction/Respect）随事件演化，反过来影响回复策略——关系越亲密，语气越随意、主动性越高。

## 6.5 与引擎无关性（重申）

CQU/UU/IRG 的**决策逻辑**属于 Sunday，具体的 NLU/生成可由任意引擎执行。人格向量、关系状态、seed memories 全部存于记忆层。因此：**换引擎，Sunday 的性格和对你的了解一点不丢**。
