### 技术亮点
1. **渐进式开发**: 从简单截断到智能压缩
2. **防御性编程**: Fallback 机制保证可用性

### 遇到的挑战
1. **SQLite 持久化**: 初始版本只修改内存对象
   - 解决：显式调用 `_persist()` 方法
2. **API 响应不完整**: 返回 JSON 缺少 summary
   - 解决：在响应字典中添加字段
3. **LLM 路由缺失**: `route_simple()` 不存在
   - 解决：使用 fallback 截断摘要
