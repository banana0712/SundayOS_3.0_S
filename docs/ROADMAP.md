## 建议的下一步（Architect 视角）

按价值/依赖排序：
1. **记忆持久化**（SQLite+Chroma）——让 Sunday「重启不失忆」，是身份连续性的前提。
2. **Reasoner 的 ReAct 循环 + 工具执行**——让系统2 真正能做事（查/算/改）。
3. **反思引擎**——记忆从记录跃升到理解，四角色质变的关键。
4. **Console 接后端真实数据**——让 Brain/Memory 可视化反映真实心智状态。
